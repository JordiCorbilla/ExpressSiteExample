ExpressSiteExample

A simple website in Node.js with Express, Stylus and Pug.

License: MIT License.

Setup

Create website folder

Copy content of repository into website folder

run the following commands:

sudo npm i
npm run build
npm start

# development mode (monitors for source code changes)
npm run dev
